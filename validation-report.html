Crypto Necessity Gut-Check — "What breaks if you remove the blockchain?"

Everything that matters. Settlement drops from T+2/days to seconds; cost from 5–7% (World Bank avg remittance) to fractions of a cent; operators hold liquidity in USDC/EURC and rebalance in seconds instead of pre-funding nostro accounts. Transparent FX (Jupiter quotes the exact route + price impact) is impossible to fake on a public ledger. Crypto is load-bearing, not ornamental.

Integration-First Assessment

This is a textbook integrate-first build — no custom on-chain program required for the MVP. That means faster to ship and near-zero audit cost.

integrating-jupiter transparent FX routing/quotes (USDC↔EURC↔local) Coinbase CDP (MCP) fiat on-ramp + KYC solana-dev wallet + RPC + transfer rails (@solana/kit) authenticate-wallet sign-in scaffold-project workspace/stack frontend-design + number-formatting the FX-transparency UI cso security audit (money app)

Custom Anchor program (solana-anchor-claude-skill) only if you later add on-chain escrow, programmatic fee splits, or recipient-routing logic — not needed for v1.
